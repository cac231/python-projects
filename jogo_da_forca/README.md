## OBJETIVO

Jogo da forca para terminal que permite escolher entre diferentes conjuntos de palavras, cada um com suas respectivas dicas.

## REQUISITOS

- Python 3.10 ou superior

## EXECUÇÃO

Pelo terminal, na pasta do projeto:

```bash
python jogo_da_forca.py
```

Ou utilizando o caminho completo:

```bash
python "caminho/do/jogo_da_forca.py"
```
