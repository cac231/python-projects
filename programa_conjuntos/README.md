## OBJETIVO

Programa para terminal que realiza diversas operações de teoria dos conjuntos utilizando listas como estrutura de dados.

## REQUISITOS

- Python 3.10 ou superior

## EXECUÇÃO

Pelo terminal, na pasta do projeto:

```bash
python conjuntos.py
```

Ou utilizando o caminho completo:

```bash
python "caminho/do/conjuntos.py"
```
