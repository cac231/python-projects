## OBJETIVO

O Tetrico é um jogo de Tetris desenvolvido com Pyxel, com uma estética inspirada no PICO-8 e visual fluido. Possui diversos modos de jogo, opções de personalização e salvamento local das partidas.

## REQUISITOS

- Python 3.10 ou superior
- Pyxel

```bash
pip install pyxel
```

## ESTRUTURA DO PROJETO

Para funcionar corretamente, mantenha os seguintes arquivos e pastas na mesma estrutura do projeto:

- `tetrico.py`
- pasta `assets`
- `my_resource.pyxpal`
- `my_resource.pyxres`

## EXECUÇÃO

Pelo terminal, na pasta do projeto:

```bash
python tetrico.py
```

Ou utilizando o caminho completo:

```bash
python "caminho/do/tetrico.py"
```
